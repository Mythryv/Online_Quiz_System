# Online_Quiz_System

## Project Description
The Online Quiz Management System is a Python project where users can log in, answer multiple-choice questions, and instantly view their final score. It demonstrates concepts like authentication, input handling, and result calculation using Python.

## Features
- User login authentication
- Multiple-choice quiz questions
- Automatic score calculation
- Percentage result display
- Modular code structure

## Technologies Used
- Python – Core programming language used to build the application
- Object-Oriented Programming (OOPS) – Used to structure and organize the code efficiently
- File Handling – For managing and storing quiz-related data
- Basic Input/Output Handling – To interact with users during the quiz

## How to Run
1. Make sure Python is installed.
2. Run: python quiz.py
